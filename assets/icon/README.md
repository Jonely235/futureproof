# FutureProof App Icon

This directory should contain your app icon file.

## Requirements:
- File name: `app_icon.png`
- Size: 1024x1024 pixels (recommended)
- Format: PNG with transparency
- Design: Should represent FutureProof/Finance theme

## Quick Icon Ideas:
1. Green checkmark + dollar sign
2. Piggy bank icon
3. Graph/chart with upward trend
4. Shield + money symbol
5. Simple "FP" logo

## How to Generate:
1. Use Canva, Figma, or Adobe Illustrator
2. Create 1024x1024 PNG
3. Save as `app_icon.png` in this directory
4. Run: `flutter pub run flutter_launcher_icons`

## For Testing:
You can temporarily use a placeholder icon or
download a free icon from IconFinder/Flaticon.

## After Adding Icon:
1. Save app_icon.png in this directory
2. Run: `flutter pub get`
3. Run: `flutter pub run flutter_launcher_icons`
4. Rebuild the app
